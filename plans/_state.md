```markdown
- **Current step**: <link>
- **Status**: in-progress
- **Owner**: <agent or human name>
- **Last updated**: <YYYY-MM-DD>
```

When you finish a step:

3. Update the **Next 3 steps** preview.
4. If a blocker exists, list it under **Blockers** with date + description.
