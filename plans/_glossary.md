# Glossary (short)

Full glossary lives in [`doc/PROJECT_DETAILS.md` Appendix F](../doc/PROJECT_DETAILS.md#appendix-f--glossary--acronyms). Below is a minimal quick reference for plan-file readers.

| Term | Meaning |
|---|---|
| HO | Head Office |
| GRN | Goods Receipt Note — inventory updates only on GRN |
| MOV | Minimum Order Value (pincode-driven) |
| POS | Point of Sale (retail billing) |
| OTP | One-Time Password |
| RBAC | Role-Based Access Control |
| RPO/RTO | Recovery Point / Recovery Time Objective |
| SRS | Software Requirement Specification |
| FY | Financial Year |
| F&F | Full & Final Settlement |
| TDS | Tax Deducted at Source |
