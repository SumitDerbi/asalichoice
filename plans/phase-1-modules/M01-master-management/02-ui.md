# M01 / 02 — UI

> **Parent**: [`index.md`](./index.md)  **Depends on**: [`01-api.md`](./01-api.md)

## Goal

Admin-UI module `src/modules/master/` with screens for every master entity. Compact, info-dense tables; drawer-form edits; bulk actions; inline-edit where safe.

## Steps

1. Register module in `module-registry.ts` with category "System / Master" and routes per entity.
2. **Pattern per entity** (use shared `<MasterListPage>` HOC):
   - List page: `<DataTable>` with search, filter chips, pagination, column visibility, export CSV.
   - Drawer create/edit using `<DrawerForm>` + TanStack Form + Zod schema co-located in `schemas/`.
   - Soft-delete via "Deactivate" toggle (not "Delete" button); reactivate from filter.
   - Bulk-deactivate, bulk-export.
3. **Specifics**:
   - **Branch**: tree view (parent/child) + flat table toggle. Edit drawer with feature-flag checkboxes.
   - **Category**: tree view with drag-to-reparent (with depth guard ≤ 4); image upload.
   - **Tax**: form lets user pick components (CGST/SGST/IGST/Cess) with rates; live "Total: X%" preview; assign HSN list.
   - **HSN**: searchable; bulk-import CSV.
   - **Payment mode**: per-branch enable matrix.
   - **Zone**: pincode-range editor (add range / individual pincodes) + map preview placeholder.
4. **Branch switcher** (in app shell from Phase 0) now reads from `GET /api/v1/master/branches/?is_active=true` via TanStack Query (`['master', 'branches', 'switcher']`). Selected branch is persisted + sent as `X-Branch-Id` header on every request.
5. **Permissions**: hide "Manage" actions when user lacks `master.manage_*`. Use `useHasPermission()` hook (stub now, real impl in M02).
6. **Command palette**: register quick actions `Create branch`, `Create category`, `Manage taxes`.
7. **Empty states + skeletons** on every list.
8. **i18n keys** from day 1 under `src/modules/master/i18n/en.json`.
9. Commit: `feat(M01): master ui screens`.

## Deliverables

- `src/modules/master/` with all entity screens.
- Branch switcher wired to real API.
- Schemas + i18n.
